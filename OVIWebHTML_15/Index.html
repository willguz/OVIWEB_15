¿Qué es Git?
Es un software de control de versiones diseñado por Linux Torvalds, pensando en la eficiencia y la confiabilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente. Git es un sistema distribuido de control de código fuente o SCM (en inglés Source Code Management).
